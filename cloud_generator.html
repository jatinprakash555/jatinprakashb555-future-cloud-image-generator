Objective

  • ✓ Develop a diffusion-based model for 0-3 hr cloud forecasting.
  • ✓ Utilize multi-channel INSAT imagery (VIS/IR/WV).
  • ✓ Create a prototype for severe weather nowcasting.
  • ✓ Validate use of indigenous satellite data for AI research.

Methodology

  1. Data Preprocessing: Align, georeference, and normalize INSAT imagery into sequences.
  2. Model Design: Implement a conditional diffusion model with a UNet backbone.
  3. Training: Optimize using a combination of MAE, MSE, and SSIM reconstruction losses.
  4. Inference & Evaluation: Generate future frames and assess accuracy with rigorous metrics.
  5. Prototype: Build this web interface for visualization and comparison.
